# «Data Universe» — корпоративный сайт IT-компании
Коммерческий адаптивный веб-сайт IT-компании на чистом HTML5/SASS и JavaScript/jQuery с семантической версткой, с перечнем услуг, карьерным блоком, клиентами компании и интерактивной картой.


## 🛠 Технологический стек

### Frontend
- **HTML5** — семантическая верстка, БЭМ-методология
- **SASS/SCSS** — модульная структура, сброс стилей, переменные, медиа-запросы
- **JavaScript/jQuery** — работа с DOM, обработка событий
- **Яндекс.Карты API** — интерактивная карта с кастомными стилями


### Библиотеки и плагины
**Owl Carousel 2** — слайдеры


## 🎯 Ключевые функции

### UI/UX элементы
- Плавная навигация по странице
- Адаптивное бургер-меню для мобильных устройств
- Адаптивные карусели (клиенты, карьера, проекты, партнеры)
- Аккордеоны для полезной информации
- Интерактивная карта с кастомными стилями


## 🔧 Технические особенности

### Адаптивность
- Адаптивный дизайн
- Медиа-запросы
- Кросс-браузерная совместимость


**Проект демонстрирует навыки разработки коммерческих корпоративных веб-сайтов IT-компании на чистом HTML5/SASS и JavaScript/jQuery с адаптивным дизайном, интерактивными элементами и интеграцией Яндекс.Карт.**
